Flower Shop Management System
Java JavaFX OOP File Handling

An automated management system for a flower boutique, featuring a dual-layer access system for Administrators and Customers. The project focuses on structured data management and user-friendly GUI.

Core Implementation:
  • Management System: Admins can manage inventory and track all customer orders in real-time.
  • Interactive UI: Developed using JavaFX with dedicated screens for Login, Registration, and Dashboards.
  • Data Persistence: Implemented a robust File Handling system to save and retrieve all store data.
  • Software Architecture: Built on Object-Oriented Programming (OOP) principles like Inheritance and Encapsulation.
